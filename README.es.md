# Bug Bounty Hunter

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Cazador de Recompensas para OpenClaw.

## Qué Soluciona
Detecta vulnerabilidades de seguridad y recompensa al descubridor.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque primero en la seguridad
- Soporte de rollback

## Uso
/bug-bounty-bot

## Ejemplo
```
Analyze code for security issues.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
